What it does

Every angle of your Guild Raid, measured.

Homina reads your guild's raid data straight from the official Tacticus API and renders it as charts and readouts your members can actually act on.

Damage & Performance

See exactly who's carrying the season — total damage, damage-per-token, and how each member stacks up against the guild average.

Get started

Up and running in three rites.

RITE 01

Add Homina

Invite the bot to your Discord server from its application page, or join the support server to try it first.

RITE 02

Register your guild

Run /register with an API key created by a (co-)leader with Guild and Raid scope. That's the whole setup.

RITE 03

Invoke the cogitator

Use any command — start with /help or /season-tokens — and Homina returns the chart in seconds.
